== Description ==
Options Framework Debugger Plugin for people who use Options Framework.

This plugin can export all Options Framework Config and show at a table, then you can know how many options you have

== Installation ==
1. Download Plugin
2. Install Plugin
3. enter The Plugin Page
5. Save Settings.


== Frequently Asked Questions ==
= should i input the options name by myself =
Normally, you don\'t need input by youself, the plugin will detect the option name
